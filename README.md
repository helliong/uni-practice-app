# Uni Practice App: IT / University Merch Store

Учебный проект интернет-магазина IT- и университетского мерча.

## Что уже сделано

- Главная страница с промо-блоками, категориями и визуальными баннерами.
- Каталог товаров с карточками, фильтрами, переключением вида и страницами товара.
- Страница товара с выбором размера/цвета и добавлением в корзину.
- Корзина с изменением количества, удалением товаров, очисткой и итоговой суммой.
- Рекомендации в корзине: при небольшом количестве товаров блок показывается горизонтально под корзиной, при большем количестве остается в сайдбаре.
- Страница оформления заказа `/checkout` с контактными данными пользователя из БД, выбором доставки и оплаты.
- Доставка на checkout: курьер, Почта России, СДЭК.
- Оплата на checkout: банковская карта онлайн, СБП по QR, карта при получении.
- Избранное с синхронизацией для авторизованного пользователя.
- Регистрация, вход и авторизация через NextAuth.
- Профиль пользователя с просмотром и редактированием имени, email и телефона.
- Синхронизация корзины и избранного с БД для авторизованных пользователей.
- Страница таблиц размеров.
- Светлая/темная тема через `ThemeContext`.

## Что используется

- **Next.js 16** с App Router.
- **React 19**.
- **TypeScript**.
- **SCSS / Sass** для глобальных и постраничных стилей.
- **Prisma** как ORM.
- **PostgreSQL** как база данных.
- **NextAuth.js** для авторизации.
- **bcryptjs** для хеширования паролей.
- **React Icons** для иконок.
- **Mock data** в `src/lib/mockData.ts` как источник товаров на текущем этапе.

## Структура проекта

- `src/app/` — страницы, layout и API routes.
- `src/components/` — переиспользуемые компоненты интерфейса.
- `src/context/` — контексты корзины, избранного, темы и провайдеры.
- `src/lib/` — Prisma client, мок-данные и утилиты.
- `src/types/` — общие TypeScript-типы.
- `public/` — изображения и статические ассеты.
- `prisma/` — схема БД и миграции.

## Локальный запуск

1. Установить зависимости:

   ```bash
   npm install
   ```

2. Создать `.env` в корне проекта:

   ```env
   DATABASE_URL="postgresql://user:password@localhost:5432/uni_practice_app?schema=public"
   NEXTAUTH_SECRET="your-secret-key"
   NEXTAUTH_URL="http://localhost:3000"
   ```

3. Применить миграции и сгенерировать Prisma Client:

   ```bash
   npx prisma migrate dev
   npx prisma generate
   ```

4. Запустить проект:

   ```bash
   npm run dev
   ```

5. Открыть `http://localhost:3000`.

## Текущее состояние

Проект уже можно запускать локально и использовать как прототип магазина: смотреть каталог, добавлять товары в корзину/избранное, редактировать профиль и переходить к оформлению заказа.

Главное ограничение сейчас — товары и оформление заказа пока не полностью завязаны на реальные сущности заказов в БД.
